# 邮件配置 {#mail}

Novaix 使用邮件发送通知，包括工单回复通知、密码重置验证码、到期提醒等。您需要配置 SMTP 服务才能使用邮件功能。

## SMTP 配置 {#smtp}

在管理面板的「系统设置」→「邮件设置」中配置 SMTP 服务信息，配置后立即生效，无需重启程序：

| 字段 | 说明 |
|------|------|
| SMTP 服务器 | 邮件服务器地址，如 `smtp.gmail.com` |
| 端口 | SMTP 端口，通常为 465（SSL）或 587（TLS） |
| 加密方式 | SSL 或 TLS |
| 用户名 | SMTP 登录用户名 |
| 密码 | SMTP 登录密码或应用专用密码 |
| 发件人名称 | 邮件中显示的发件人名称 |
| 发件人地址 | 邮件中显示的发件人邮箱 |

::: tip
常用邮件服务商的 SMTP 配置：
- **Gmail**：`smtp.gmail.com`，端口 587，需要使用[应用专用密码](https://support.google.com/accounts/answer/185833)
- **阿里云企业邮箱**：`smtp.qiye.aliyun.com`，端口 465
- **腾讯企业邮箱**：`smtp.exmail.qq.com`，端口 465
:::

::: warning 注意
未配置 SMTP 时，所有需要发送邮件的功能（验证码、密码重置、工单通知、到期提醒等）都会**静默跳过**，不会报错也不会有任何提示。如果您发现用户收不到验证码或通知邮件，请先检查 SMTP 是否已正确配置。
:::

## 邮件模板 {#templates}

Novaix 内置了多种邮件模板，您可以在「系统设置」中自定义模板内容。系统会自动替换模板中的变量（如用户名、验证码、实例信息等）。

## 邮件接收（Inbound） {#inbound}

Novaix 支持通过接收邮件来自动回复工单。用户直接回复工单通知邮件，邮件内容会自动添加到对应工单中。

该功能需要配置邮件接收的 Webhook 地址：

```
https://您的域名/api/callbacks/mail/inbound
```

具体的配置方式取决于您使用的邮件服务商，通常在邮件服务商的「Inbound Parse」或「Webhook」设置中配置。
