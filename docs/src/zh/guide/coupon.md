# 优惠券 {#coupon}

Novaix 支持灵活的优惠券系统，您可以创建各种类型的优惠券来促进销售。

## 创建优惠券 {#create}

在管理面板的「优惠券管理」中创建优惠券，支持以下配置：

### 基本配置 {#basic}

| 字段 | 说明 |
|------|------|
| 优惠券码 | 用户在下单时输入的兑换码 |
| 类型 | 固定金额减免 或 按百分比折扣 |
| 折扣值 | 减免金额或折扣百分比 |

### 使用限制 {#limits}

| 字段 | 说明 |
|------|------|
| 总使用次数 | 所有用户累计可使用的总次数 |
| 每人使用次数 | 单个用户最多可使用的次数 |
| 最低订单金额 | 订单金额达到多少才能使用 |
| 最大折扣金额 | 百分比折扣时的最大减免上限 |
| 生效时间 | 优惠券的生效开始时间 |
| 过期时间 | 优惠券的过期时间 |

### 适用范围 {#scope}

您可以限制优惠券适用的订单类型，例如仅限新购订单使用，或仅限续费订单使用。

## 使用统计 {#stats}

在优惠券列表中，您可以查看每个优惠券的使用次数和剩余可用次数。

## 注意事项 {#notes}

::: warning 注意
- **优惠券码区分大小写**。建议统一使用大写字母和数字，避免用户输入时因大小写问题而无法使用。
- 优惠券创建后，优惠券码不可修改。如果需要更换码值，请创建新的优惠券。
- 如果优惠券设置了「最大折扣金额」，当百分比计算出的折扣超过此上限时，实际折扣以上限为准。例如：10% 折扣 + 最大折扣 50 元，用户购买了 1000 元的套餐，折扣应为 100 元，但实际只减免 50 元。
- 已使用的优惠券即使被删除，也不会影响已生成的订单。
- 如果您想暂停某个优惠券的使用，可以将其过期时间设为当前时间之前，而不必删除它。
:::
