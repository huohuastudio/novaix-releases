# 代理系统 {#agent}

Novaix 内置了代理商（分销商）系统，您可以通过代理模式拓展销售渠道。代理商通过推广链接引导用户注册和下单，成功交易后可以获得佣金。

## 设置代理商 {#setup}

管理员可以在「用户管理」中将用户角色修改为「代理商」，该用户就会获得代理功能。

## 代理商功能 {#features}

代理商在用户面板中可以：

- 查看自己的专属推广链接
- 查看通过推广链接注册的用户列表
- 查看佣金记录和佣金总额
- 查看推广带来的订单情况

## 佣金计算 {#commission}

当通过代理商推广链接注册的用户完成支付后，系统会自动计算佣金并记录到代理商的佣金账户中。佣金比例由管理员在系统设置中配置。

## 佣金管理 {#management}

管理员可以在「代理管理」中查看所有代理商的佣金记录，进行佣金结算和管理操作。
