# 工单系统 {#ticket}

Novaix 内置了工单系统，用户可以通过工单与管理员沟通，提交问题或请求。

## 用户端 {#user}

用户可以在用户面板中：

- 创建新工单，选择优先级（低、中、高）
- 查看自己提交的工单列表和状态
- 在工单中回复消息，与管理员对话

## 管理员端 {#admin}

管理员可以在管理面板中：

- 查看所有工单，按状态筛选
- 回复工单
- 关闭工单

## 工单状态 {#status}

| 状态 | 说明 |
|------|------|
| 待处理 | 用户新建的工单，等待管理员处理 |
| 已回复 | 管理员已回复，等待用户确认 |
| 用户回复 | 用户有新的回复，等待管理员处理 |
| 已关闭 | 工单已结束 |

## 邮件通知 {#email}

工单系统支持邮件通知功能。当工单有新回复时，系统会自动发送邮件通知对方。此功能需要先配置好 [SMTP 邮件服务](./mail)。

Novaix 还支持通过邮件回复工单。用户直接回复通知邮件的内容会自动添加到对应工单中，无需登录面板即可沟通。

::: tip
邮件回复功能需要配置邮件接收（Inbound）服务，详见[邮件配置](./mail)。
:::
