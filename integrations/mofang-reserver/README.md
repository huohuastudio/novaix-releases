# 魔方 V10 — Reserver（上游供应商）模块

将 Novaix 作为智简魔方 V10 的"上游供应商"接入，实现商品自动同步和加价转售。

> 后端兼容 API + PHP 控制器 + 前台 Vue 模板已实现。购买页为固定配置套餐，无自定义交互模板。需搭建魔方 V10 测试环境进行端到端验证。

## 安装

将 `novaix/` 目录复制到魔方 V10 安装目录：

```
public/plugins/reserver/novaix/
```

魔方后台添加供应商后也会自动下载安装此模块。

## 配置

在魔方后台 → 供应商管理 → 添加供应商：

| 字段 | 填写内容 |
|------|---------|
| 类型 | `默认业务系统` |
| 名称 | 自定义名称 |
| 接口地址 | `https://your-novaix.com/compat/mofang`（注意路径后缀） |
| 用户名 | 任意（如 `api`） |
| API 密钥 | Novaix 管理面板创建的 API 密钥（`nv_` 开头，需 `provision` 读写权限） |

## 支持的功能

- 商品自动同步
- 自动开通
- 暂停/解除暂停
- 删除（退款）
- 续费
- 开机/关机/重启
- 重装系统
- 重置密码
- VNC 控制台
